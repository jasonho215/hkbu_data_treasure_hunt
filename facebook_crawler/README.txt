prerequisite
1.  setup a facebook developer account
2.  setup a facebook application
3.  enable "facebook login" in product category
4.  in "facebook login", add "localhost:PORT_NUM" to "Vaild OAuth redirect URL" field. by default, the PORT_NUM is 8080.

software needed:
1.  npm
2.  node.js engine
3.  visual studio code
4.  npm package -- http-server by "npm install -g http-server"

note:
1.  Open http-server in current directory by terminal before run the crawler 
2.  do not close the terminal when you fetching data.
3.  Allow pop-up windows in browser
4.  Check disable cache in web browser is valid
